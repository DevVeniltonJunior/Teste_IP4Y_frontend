# IP4Y

Teste_Ip4y/Front-end

## Configuração

1. Clone o repositório.

```bash
git clone https://github.com/DevVeniltonJunior/Teste_IP4Y_frontend.git
cd Teste_IP4Y_frontend
```

2. Instale as dependências.

```bash
npm install
```

3. Inicie o servidor de desenvolvimento.

```bash
quasar dev
```

## Uso

Funcionalidades:

  - Criar Usuário
  - Atualizar Usuário
  - Deletar Usuário
